CIRCUIT CONNECTIONS

Component Connected To Connection Reason
PIR-SESNOR ESP-32 VCC → 3.3V Powers the PIR sensor to detect motion.
ESP-32 GND → GND Provides a common ground reference.
ESP-32 OUT → D4 Sends a HIGH signal to ESP8266 when motion is detected.
IR-SESNOR ESP-32 VCC → 5V Powers the IR sensor for detecting objects.
ESP-32 GND → GND Provides a common ground reference.
ESP-32 DAT → D5 Sends a signal to ESP-32 when an object is detected.
LED ESP-32 POSITIVE (ANODE) → D18 The ESP-32 controls the LED by providing voltage to turn it ON.
ESP-32 NEGATIVE (CATHODE) →330Ω resistor → GND The resistor limits the current to prevent LED damage.
