TITLE:Automatic Street Light Control System using Wireless Sensor Networks


YEAR OF PUBLICATION:2017


METHODOLOGY:

For implementing Automatic Street Lighting, this paper involves method which is a combination of sensor-based control, microcontroller automation, wireless communication, and power optimization to create a smart street lighting system. These methods improve efficiency, reduce operational costs, and make urban infrastructure more sustainable.


  • Sensors used here to build automatic street lighting system are

    1. Light Dependent Resistor (LDR): It Measures ambient light levels. If the intensity drops below a threshold (e.g., evening time), the lights turn on.

    2. PIR (Passive Infrared) Sensors: Detects motion of pedestrians or vehicles. If motion is detected, the lights turn on; otherwise, they remain off or dimmed to save energy

  • A microcontroller (such as an Arduino) processes data from the LDR and PIR sensors. It makes the decisions to turn lights on or off based on sensor inputs, eliminating the need for manual operation. The microcontroller also controls dimming to reduce power consumption when no movement is detected.

  • The system uses a wireless sensor network (WSN) to connect streetlights and enable remote control. Each streetlight node communicates with others to optimize lighting across an entire area. This ensures that if motion is detected at one point, the adjacent lights may also turn on to improve visibility.

TITLE:Automatic Light System Using LDR and PIR Sensor


YEAR OF PUBLICATION:2024


METHODOLOGY:

  • When the LDR detects low light levels, the system prepares to turn on the light

  • If the PIR sensor detects human motion, the light is turned on.

  • If no motion is detected for a set time, the light turns off to save energy

  • The system is programmed using the Arduino IDE, and code execution is based on sensor inputs

  • The system was tested using Tinkercad, an online simulation tool

  • Various scenarios were evaluated to measure sensor response, accuracy, and energy efficiency

PAPER-3

TITLE:ENERGY SAVING STREET LIGHT USING IOT BASED ON MOTION DETECTION


YEAR OF PUBLICATION: 2023


METHODOLOGY:

The proposed system in the paper is an IoT-based smart street lighting system that uses sensors to control streetlights dynamically based on motion detection and ambient light levels.

  1. Automatic Brightness Adjustment

    • The LDR sensor detects sunlight levels and ensures that streetlights turn on at sunset and off at sunrise.

    • The IR sensor detects motion and turns the lights on only when a person or vehicle is nearby.

    • If no movement is detected, the lights remain dimmed to save energy.

  2. IoT-Based Monitoring & Control

    • Authorities can check the status of streetlights in real-time and receive alerts in case of faults.

  3. Dimming for Energy Efficiency

    • The system gradually reduces brightness during late-night hours when traffic is minimal.

    • This reduces energy consumption and extends the lifespan of the LED lights

  4. Communication and Software

    • Arduino IDE is used for programming the microcontroller.

    • IoT platform is used for real-time data monitoring and analysis
