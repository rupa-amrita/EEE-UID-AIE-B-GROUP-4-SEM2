Methodology

  • We use three IR (Infrared) sensors to detect any movement of people or vehicles near the street lights.
  • Each IR sensor is linked to an individual LED light (street light).
  • When movement is detected, the corresponding LED turns on automatically.
  • An LDR (Light Dependent Resistor) checks the surrounding light levels.
  • The lights only turn on at night or in darkness — during the day, they stay off even if motion is detected.
  • The brightness of each LED can be customized by the user through the Serial Monitor (in Arduino IDE).
  • Users can enter three brightness values (between 0 to 255), one for each light.
  • If no user value is given, the lights glow at full brightness (255) when triggered.
  • If the user types "reset" or "clear", the custom brightness is removed and the system uses default behavior.
  • This system saves electricity by only turning on lights when needed and giving users control over brightness.
  • The program runs continuously in a loop, checking sensors, light levels, and user input in real-time.
